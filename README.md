# Cam_Feild_Setting_UI
an UI that can do
1. camera distortion calibration
2. camera-to-real_world projection by perspective tramsform
3. polygon painting on camrea view

## Calibration
> python realtime.py


每三秒擷取一張frame，且用累計下來的frame去做calibration
